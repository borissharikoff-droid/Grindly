## SCENE BREAKDOWN

### [0:00–1.5s] HOOK — Pattern interrupt
Rotating hooks:
- Text overlay on gameplay: "I turned my work hours into XP and now I can't stop"
- POV shot of XP bar leveling up mid-coding session
- Stat screen: "8 hours deep work → Level 34 Developer"
- Split: Left = doomscrolling. Right = Grindly user leveling up

### [1.5s–12s] PROBLEM — Relatable pain
B-roll / stock footage:
- Staring at blank code editor, procrastinating
- Checking phone every 5 minutes
- Closing laptop in frustration
- Discord notifications destroying focus

Subtitle: "Every programmer knows this feeling."  
VO: Short, punchy. Max 2 sentences.

### [12s–35s] SOLUTION — App gameplay footage
Actual Grindly screen recordings:
- Timer running while VS Code is active
- XP bar filling up in real-time
- Skill level up animation (pop-up notification)
- Inventory with loot dropped from a session
- Friends leaderboard — "your friend just hit level 40"
- Pet companion reacting

Subtitles on every line.  
VO or text-only narration — "You earn XP just by doing the work."

### [35s–45s] PROOF / SOCIAL — Credibility insert
One of:
- Discord screenshot: "I worked 6 hours straight because of this app"
- Leaderboard showing real usernames + XP
- Before/after screen time stats
- "10,000+ users grinding daily"

### [45s–55s] CTA — Direct, one action
"Download Grindly free on Windows → link in bio"  
Text overlay only. No talking. Big font, 2 seconds hold.

---

## AUDIO RULES (mandatory)

- **Background:** lo-fi beats, chiptune RPG OST, or synthwave — NOT copyright
- **VO (optional):** calm male or female voice, no hype, matter-of-fact tone
- **SFX:** UI click sounds, level-up chime, XP ding synced to gameplay
- **All speech subtitled:** white text, black drop shadow, center bottom

---

## SUBTITLE RULES (mandatory)

- **Font:** Bold, clean sans-serif (Montserrat or similar)
- **Size:** Large enough to read on mobile without zooming
- **Position:** Lower third, never blocking key visuals
- **Auto-caption everything**, then proofread for accuracy
- **Animate word-by-word** if platform supports it (CapCut / Premiere)
