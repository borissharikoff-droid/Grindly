## What's new in v4.6.0

**New**
- Group chats: create groups, invite friends, send messages, roll dice (🎲 0–100)
- Friends page redesign: Friends / Groups tabs, unread indicators, last message preview
- Group reactions, member avatars, kick/leave/delete with 2-click confirmation
- 20 new item skins: Lich Set, Titan Set, Wolf Fang Pendant, Warlord Gauntlets, 3 Raid items, Zone 7–8 materials, Wilted Plant

**Improved**
- Character panel: item icons are larger and clearer
- Arena requirements section is now collapsible
- Auto-farm: death penalty no longer voids mob gold earned mid-run

**Fixed**
- Dungeon boss loot: switching tabs mid-dungeon no longer bypasses the chest-open animation
- Level cap: XP bar no longer shows NaN at level 99
- Legendary chest was showing epic chest image — fixed
- `isAutoRunning` no longer persists after a crash (auto-mode can't get stuck)
- `clear_all_zones` achievement target corrected (6 → 8 zones)
- What's New modal now only appears after daily login claim completes
