## What's new in v4.16.0

**Abyss Descent — Endless Delve Mode**

- Delve mode — endless combat. Multiple mobs per floor; every 10th is a boss; every 9th is rest.
- Hardcore vs Casual — casual returns loadout safely; hardcore stakes gear + items.
- Stake modal — pick equipped gear + up to 8 inventory slots + 4 food slots before each run.
- Multi-mob real-time combat — 2–5 enemies on screen with independent HP. ATK splits (CLEAVE).
- Weekly Hardcore leaderboard — top 10 deepest descents each week.
- 8 Abyss cosmetic skins unlocked at floor milestones (10/25/50/75/100/150/250/500 HC).
- Warrior XP boosted in Delve — finally a viable path to lvl 99.
- Floor mutators from floor 15+ — Glass Cannons, Swarm, Fortified.
- Arena tab auto-locks during a Hardcore run — prevents accidental run loss.
- Hardcore mode tints whole app warrior-red.
- Refactored arena stat composition into composePlayerSnapshot helper.
